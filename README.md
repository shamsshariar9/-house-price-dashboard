# 🏠 House Price Prediction Dashboard

## 📌 Project Overview
This project is a Machine Learning-based web application that predicts house prices using Linear Regression. The dashboard is built using Streamlit.

---

## 📊 Dataset
The dataset includes features such as:
- RM (average number of rooms)
- LSTAT (lower status population percentage)
- MEDV (median house value)

---

## ⚙️ Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 🤖 Machine Learning Model
A Linear Regression model is trained to predict house prices based on selected features.

---

## 📈 Features
- Interactive sliders for input
- Real-time prediction
- Data visualization
- Actual vs Predicted graph
